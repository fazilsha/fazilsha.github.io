

# Portfolio Redesign — Fazil Sha, Cybersecurity Professional

## Design
- **Dark & Minimal theme** with subtle glow accents, clean typography, smooth scroll animations
- Fully responsive (mobile, tablet, desktop)
- Single-page layout with sticky navigation

## Sections to Build

### 1. Sticky Navigation Bar
- Name/logo on left, section links on right, mobile hamburger menu
- Smooth scroll to each section

### 2. Hero Section
- Name + "Cybersecurity Professional" with typing/fade animation
- Profile photo with subtle glow effect
- Social links (LinkedIn, GitHub, Discord, Facebook, Instagram)
- "Get in Touch" CTA button

### 3. About Me
- Detailed bio about your cybersecurity journey and passion
- Clean layout with decorative accent elements

### 4. Skills
- Visual skill cards for: **Network Pentesting, Web App Pentesting, Cloud Security, LLM Security**, plus supporting skills (Python, Linux, AWS, Networking)
- Categorized into specializations vs tools/platforms

### 5. Projects / Activities
- Card grid showcasing projects from your current portfolio (AmiPwned, Python basics, AWS boto3, etc.)
- Each card with title, description, and GitHub link
- Hover glow effects

### 6. Certifications & Education
- Timeline/card layout with placeholder content for you to fill in

### 7. Blog / Write-ups
- Article cards with placeholder content (title, date, excerpt)
- Ready for you to add real write-ups later

### 8. Contact Section
- Contact form (name, email, message) — frontend-only for now
- Social links repeated for alternative contact

### 9. Footer
- Copyright, quick nav links, social icons

## Technical Approach
- All content hardcoded (no backend needed)
- Scroll-triggered fade-in animations using CSS/Intersection Observer
- Dark color scheme via Tailwind CSS custom theme variables
- Profile photo referenced from your existing GitHub page
